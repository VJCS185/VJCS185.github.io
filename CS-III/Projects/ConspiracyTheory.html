The Basics of Recommendation Algorithms

   Recommendation algorithms are used by social media companies such as Meta and TikTok to keep the user on the application for as long as possible. This is done by collecting information on the viewer's interest and constantly bombarding them with content they are likely to love.
   First, the user interaction data is collected. This includes likes, saves, comments, shares, and watch time. Since the user will have more interactions with certain videos they like, the recommendation algorithm can build a profile on the types of content the user likes. Then, the algorithm uses this information and analyzes other content to find the content the user will like. Then it feeds this content to the user keeping them on the app for as long as possible each session.


Feedback Loop

   The feedback loop is part of the recommendation algorithm that makes sure to feed the best content to the user. The feedback loop allows the algorithm to update the user profile as the users become interested in different types of content.
   First, the algorithm collects data on which posts the user interacts with. The interactions show the user likes the content. Then the algorithm builds a profile on the user by storing their interaction data. After that, the data is used to find content that is similar to the content the user likes, and it feeds the content to the user.


Conspiracy Theories & Algorithmic Amplification

   Conspiracy Theories are controversial ideas or speculations that are spread. This type of content usually has lots of interactions, meaning many people like the content, because it is highly engaging as it acts like a story with a build-up. This leads to many user profiles being associated with conspiracy theory content and the algorithm continues to feed the user with this content to retain the user’s attention.
   This leads to many users being fed lots of misinformation. With these social media platforms having billions of users, many of whom show interest in conspiracy theory content, the spread of misinformation can have a significant effect on the thoughts of the population.


Platform Incentives

   Companies want users to stay on the application for as long as possible because this allows more advertisements to be fed to the user. This is because advertisements are paid by the number of views they get. These companies are only concerned with increasing their profits and have little to no concern for the spread of misinformation. This leads to the algorithm working solely in the favor of keeping users on the platform without regard for misinformation. With billions of users, this can create a significant impact on the beliefs of the population.


Potential for Exploitation

   Since the content shared is not heavily regulated for the control of misinformation, Malicious users who understand how the recommendation algorithm works can spread misinformation that receives lots of interactions. This allows the malicious user to create content that has a lot of reach and also spreads misinformation. These malicious users can affect the thoughts of those they influence through their videos and the algorithm is incentivized to spread their content. This leads to these videos with misinformation going viral and affecting many people who use the platform.
